# SCHEDULING OPTIMISATION README

scheduling_optimisation.zip uses a scheduling heuristic with ILP to find the optimal set of drone items.

There are 2 scripts: 
1)Scheduling_heuristic.py 
2)ILP_optimisation.py

List of healthcare facilities: 
1)optimised_journeys_94.txt

## Installation

Download scheduling_optimisation.zip from `https://github.com/al4517/drone-delivery`

## Usage

1)Scheduling_heuristic.py = Determines the delivery time, delay time and spare time of scheduling deliveries for a set of drone items.

2)ILP_optimisation.py = Finds the optimal drone item set that minimises costs and delay time
